# README - Como Rodar o Projeto

Este projeto contém dois componentes principais: 
o **cliente** (uma aplicação React) e 
o **servidor** (uma aplicação Node.js). 

Abaixo estão as instruções para configurar e rodar ambos os projetos.

## Passos para Rodar o Cliente (React) e o Servidor (Node.js)

1. **Configuração do Cliente (React)**

    - Navegue até a pasta do cliente:
      Abra o terminal e execute o comando:
      ```
      cd cliente
      ```

    - Instale as dependências do cliente:
      Execute o comando:
      ```
      npm install
      ```

    - Inicie o servidor de desenvolvimento do cliente:
      Execute o comando:
      ```
      npm start
      ```
      Isso iniciará o projeto React e o cliente estará disponível em `http://localhost:3000`.

2. **Configuração do Servidor (Node.js)**

    - Navegue até a pasta do servidor:
      Abra um segundo terminal e execute o comando:
      ```
      cd servidor
      ```

    - Instale as dependências do servidor:
      Execute o comando:
      ```
      npm install
      ```

    - Inicie o servidor Node.js:
      Execute o comando:
      ```
      node app.js
      ```
      O servidor ficará disponível em `http://localhost:3001`
